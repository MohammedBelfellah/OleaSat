05 — AI / Data Pipeline
From Pixels to Prediction
Step 1 — GEE Sentinel-2 Extraction
import ee import geemap ee.Initialize() def compute_indices(image): ndvi = image.normalizedDifference(['B8', 'B4']).rename('NDVI') ndwi = image.normalizedDifference(['B8A', 'B11']).rename('NDWI') return image.addBands([ndvi, ndwi]) def extract_parcel_timeseries(geojson, start, end): aoi = ee.Geometry.Polygon(geojson['coordinates']) col = (ee.ImageCollection('COPERNICUS/S2_SR_HARMONIZED') .filterBounds(aoi) .filterDate(start, end) .filter(ee.Filter.lt('CLOUDY_PIXEL_PERCENTAGE', 20)) .map(compute_indices)) # Reduce to mean per image per polygon return col.getRegion(aoi, scale=10).getInfo()
Step 2 — Feature Engineering
# After merging satellite + meteo DataFrames def build_features(df): # Lag features (what was NDVI 5, 10, 15d ago?) for lag in [5, 10, 15, 30]: df[f'ndvi_lag_{lag}'] = df['ndvi'].shift(lag) # Rolling stats df['rain_14d'] = df['rain_mm'].rolling(14).sum() df['et0_7d'] = df['et0'].rolling(7).mean() df['days_no_rain'] = (df['rain_mm'] == 0).cumsum() # Seasonal anomaly df['ndvi_anom'] = df['ndvi'] - df.groupby('doy')['ndvi'].transform('mean') return df.dropna()
Step 3 — Ground Truth Strategy
⚠️ The Critical Missing Piece

Without labeled soil moisture data, the model can't be validated. Use one of these strategies:


Option A — Physical Ground Truth

Partner with 1 pilot farm → install 2-3 cheap capacitive probes (≈ 30€ each) → collect 3 months of real SWC data → use to train and validate model.

Option B — Water Balance Proxy

Compute daily Kc×ET₀ (FAO-56 method). Use cumulative water balance as label. No physical sensor needed — proven agronomic model.

Option C — SMAP + ISMN Dataset

Use NASA SMAP L4 (9km) or ISMN station data for Morocco region. Pre-existing labeled dataset. Less spatially precise but works for MVP.
